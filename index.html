O que são Tags?

O HTML para se comunicar com o navegador utiliza de uma estrutura característica da linguagem de marcação, que nada mais são do que as Tags, as Tags são definidas por duas características, as Tags com abertura e fechamento, e também as tags que são automaticamente os dois, ou seja, ela abrem e fecham na mesma Tag, o nome dado para isso é self-closed, assim a tag de abertura fica com o sinal da barra (/) fechando automaticamente a Tag.

O que são Atributos?

atributos são características que você pode adicionar nas tags de abertura do HTML.
Estes atributos adicionados podem ser de diversas finalidades, como: para identificação (ID), para definições de campo de um objeto como altura e largura (width e heigth), para criar um definição tipo para a Tag (type), para linkar uma Tag a um site, imagem ou a outra Tag e etc(href). Enfim com os atributos e possível modificar, manipular e alterar o comportamento ou característica de um Tag.
